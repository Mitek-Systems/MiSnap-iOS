  • recordVideo

    When set to TRUE a video recording of a session (no sound) is returned. When set to FALSE a session is not recorded

    Default: FALSE.

    Declaration

    Objective-C

    @property (nonatomic) BOOL recordVideo;
  • recordAudio

    When set to TRUE and recordVideo set to TRUE a video recording of a session with sound is returned. When set to FALSE audio is not recorded

    Default: FALSE.

    Declaration

    Objective-C

    @property (nonatomic) BOOL recordAudio;
  • videoQuality

    Specifies relative quality of a recorded video

    Range: [10, 100]

    10 - video is recorded with a minimum supported bitrate (the smallest size with a decent quality)

    100 - video is recorded with a highest supported bitrate (the biggest size with the best quality)

    Default: 10

    Declaration

    Objective-C

    @property (nonatomic) NSInteger videoQuality;
  • showRecordingUI

    Show recording UI

    Values: 0, 1

    Key: kMiSnapShowRecordingUI

    Note

    Only used when recordVideo parameter is TRUE

    Declaration

    Objective-C

    @property (nonatomic) BOOL showRecordingUI;
  • scanBarcode

    Indicates whether a camera should be configured to scan barcodes

    Default: FALSE.

    Note

    It’s recommended to override to TRUE if your app supports only AAMVA PDF417 barcodes (US and Canada driver licenses) as there’s a defect in Apple’s API where non-English alphabet characters are decoded inacurately (e.g. Spanish and other international DLs)

    When overridden to TRUE only PDF417 barcode is being detected by default but list of supported barcodes can be modified by overriding supportedBarcodeTypes property

    Declaration

    Objective-C

    @property (nonatomic) BOOL scanBarcode;
  • supportedBarcodeTypes

    An array of MiSnapCameraBarcodeType a camera should scan for

    Declaration

    Objective-C

    @property (nonatomic) NSArray *_Nonnull supportedBarcodeTypes;
